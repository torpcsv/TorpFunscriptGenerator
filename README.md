# TorpFunscriptGenerator

## 概要
動画・音声に合わせて電動グッズの動作を制御する

funscriptファイルを生成するWindowsアプリケーションです。

!["TorpFunscriptGenerator"](/images/TFG_v1.0.0.png)

## 動作環境
- Windows 11
- .NETFramework,Version=v4.8

## インストール方法
1. Releasesからダウンロード・解凍してexeを実行してください。

## 依存関係
本ソフトはMicrosoft Windowsに含まれるコンポーネント（Windows Media Player）を利用しています。
